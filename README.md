Download these 3 files, create a new project with PyCharm call it "PDF_Reader"
Copy the 3 files and paste them inside your PyCharm project
Check out main.py and pdf_reader.py (import statements) and if you are missing the required libraries download/install them
Right click/select pdf_reader.py and run it
The first parameter you have to enter is the exact location on the .pdf file you want to read (already present in your project)

After you enter the correct location of the pdf file you want to read/process a menu with 7 options will be displayed, 
select one of the option and follow the instructions on the screen

Enjoy 8-)
